
Smart Library Management System

A Smart Library Management System designed to efficiently manage books, students, issue/return records, and library operations. This project helps libraries automate daily tasks and maintain records in an organized and user-friendly way.

Features
Add, update, and delete books
Student/User management
Issue and return book system
Search books by title, author, or category
Track available and issued books
Admin dashboard and authentication
Responsive and easy-to-use interface
Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Node.js / JavaScript
Database: SQL
